Propositional vs. FOL encoding
a. For rule count scaling with an n x n grid size, prpositional encoding would require at least a n x n number of rules to assign to each grid square, in this case 3n^2. In FOL encoding, since a single rule is constant for the entire grid, this case would only require 3 rules.

b. FOL encoding is more readable due to both its compact and structured form and how its easier to associate object/rule relations.

c. Domain closure is needed for FOL encoding to enclose the boundaries when solving. Propositional encoding does not require it since each individual state is identified and fixed.
